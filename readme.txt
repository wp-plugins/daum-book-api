== Description ==

DAUM BOOK API is the plugin that add book information in post use daum book API.
You enter API key and then write short code in post.
In post, display image, title, data about book.
This plugin is useful to Korean. 

Major features in DAUM BOOK APIt include:

* shortcode : write [daumbook]
* API Setting 
* display image, title, date, author about book

== Installation ==

1. Upload the DAUM BOOOK API plugin to your blog, Activate it
2. In Settings -> DAUM BOOK API Settings, enter your API Key
3. In post, write [daumbook] shortcode
